  3D visualisation


museum zuidas

For BSc we had to model different alternatives for the Zuidas Museum in Maya. It took me a while to finish this project, because my Maya-skills weren't that... well.. I didn't have them. In the meantime I got to know the program a little better.


3D computer visualisation in an architectural design environment

This course helped me to figure out how Maya works. The assignment was to make one huge render (A1). That sounds pretty easy, but it actually isn't. I learned to model, create shaders and materials, etc. The end result is quite nice I think.


space customizer

For this course I had to work with Rhino; an other program used for 3D visualisation. It looks a lot like Maya, but it is easier to understand. I must warn everyone though, working with different kind of 3D visualisation programs can become a quite confusing...
